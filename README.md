# Тесты по русскому языку · 5–11 классы

Интерактивные тесты к вводным урокам о русском языке — **«Учительская» / Арина Галицкая**.

**Онлайн:** https://prouchitelskaya.github.io/russian-tests-5-11/

## Что внутри

Одна страница — семь блоков тестов, по одному на класс. В каждом классе **20 вопросов** с мгновенной проверкой и объяснением каждого ответа.

| Класс | Тема |
|---|---|
| 5 | Богатство и выразительность русского языка |
| 6 | Русский язык – один из развитых языков мира |
| 7 | Язык как развивающееся явление |
| 8 | Русский язык в кругу других славянских языков |
| 9 | Международное значение русского языка |
| 10 | Слово о русском языке |
| 11 | Язык как явление национальной культуры |

Прямые ссылки на класс: `…/#g5`, `…/#g6`, … `…/#g11`.

## Технически

- Статический сайт: HTML + CSS + vanilla JS, без сборки и зависимостей.
- `data.js` — банк вопросов (JSON). Авторская конвенция: правильный ответ всегда `options[0]`, `answer: 0`; на странице варианты перемешиваются.
- Прогресс (лучший результат по классу) хранится в `localStorage` (`rustests511:v1`).
- Светлая/тёмная тема — по `prefers-color-scheme`.
